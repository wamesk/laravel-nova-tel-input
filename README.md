# Laravel Nova Tel Input


<img src="img.png">


## Installation

``` php
composer require wamesk/laravel-nova-tel-input
```

## Usage

``` php
use Ppp\TelInput\TelInput;

TelInput::make(__('phone'), 'phone'),

TelInput::make('tel', 'tel')->required()->rules('required')
    ->showDialCode(true)
    ->preferredCountries(['SK', 'CZ'])
    ->onlyCountries([]),
```
